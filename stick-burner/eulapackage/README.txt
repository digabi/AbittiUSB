EULA && ClickOnce
=================

Jotta saat EULA:n (End User License Agreement) clickOnce:n asennukseen, kopioi
tämän kansion kamat bootstrapper kansioon kansioon ja käynnistä Visual Studio
uudelleen.  Kansion oikea paikka löytyy kenties registrystä avaimesta:

  HKLM\Software\Wow6432Node\Microsoft\GenericBootstrapper\11.0

Lisätietoa:
- https://docs.microsoft.com/fi-fi/visualstudio/deployment/creating-bootstrapper-packages
- http://stackoverflow.com/questions/834071/clickonce-ask-for-a-license-agreement
